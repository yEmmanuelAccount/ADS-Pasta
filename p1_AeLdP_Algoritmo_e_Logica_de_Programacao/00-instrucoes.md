# Como compilar pelo terminal
````
gcc q[numero_da_questao].c -o q[numero_da_questao]
````

# Como compilar pelo terminal (usando biblioteca externa)
````
gcc q[numero_da_questao].c -o q[numero_da_questao] -l[nome_da_biblioteca]
````

# Como rodar o arquivo pelo terminal
````
./q[numero_da_questao]
````
