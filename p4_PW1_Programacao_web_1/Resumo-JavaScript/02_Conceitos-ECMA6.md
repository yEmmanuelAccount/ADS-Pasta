<h3 class="pw1"> ECMAScript </h3>

Usando o **ECMAScript**, é possível:
* criação de novos tipos de dados (exemplos: **Map**, **WeakMap**, **Set**, **WearSet**)
* declaração de variáveis como `let` (recebe qualquer coisa) e `const` (constante).
