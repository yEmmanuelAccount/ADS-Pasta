Introdução a LSW

História e Evolução da Web

A Web surgiu da necessidade de organizar dados que antes viviam isolados, evoluindo de documentos estáticos para redes inteligentes e semânticas.

As Eras da World Wide Web (WWW)

Web 1.0 - Meramente Informativa

Focada em empresas e leitura. Caracterizada por documentos estáticos e formulários simples em uma arquitetura Cliente/Servidor clássica.

Web 2.0 - Colaborativa e Social

Marcada pela bi-direcionalidade. O conteúdo passa a ser gerado pelos usuários (Facebook, YouTube, Wikipedia), focando na comunidade e no dinamismo.

Web 3.0 - Semântica

Extensão onde a informação tem significado definido para máquinas e pessoas. O objetivo é transformar a web de documentos em uma web de dados estruturados.

Web 4.0 - Inteligente

Integração de Inteligência Artificial (IA), Metaverso e Internet das Coisas (IoT) para tornar a rede mais autônoma e preditiva.

Conceitos Básicos de Arquitetura Web

Diferenciar o ambiente (Web) da infraestrutura (Internet) é fundamental para entender como os sistemas se comunicam.

Arquitetura Cliente/Servidor

O cliente faz uma requisição e o servidor processa e devolve uma resposta.

Camadas de Execução
  • Client-side: Aplicações que rodam no navegador do usuário.
  • Server-side: Processamento realizado nos computadores servidores.
Identificação e Comunicação

Cada recurso possui uma URL (endereço único). A comunicação é regida pelo Protocolo HTTP, utilizando métodos como:

  • GET: Solicita um recurso.
  • POST: Envia dados ao servidor (ex: formulários).
  • HEAD: Solicita apenas os cabeçalhos da resposta.

Interfaces e Padrões

API (Application Programming Interface)

Normas que permitem a comunicação entre plataformas. Padrões comuns incluem REST (verbos HTTP) e GraphQL (linguagem de consulta específica).

Arquitetura de Software

Decisões de projeto que visam reusabilidade e coesão, utilizando padrões como MVC ou Arquitetura Limpa.

Roadmap do Desenvolvedor Web

Roteiro sugerido para guiar o aprendizado com foco em produtividade e fundamentos sólidos.
