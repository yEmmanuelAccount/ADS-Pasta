Obtendo IP do CLIENTE

  • Abra o menu Iniciar, depois clique em Acessórios e depois em Terminal.
  • Digite ifconfig e procure o IP no campo eth0.
  • IP-Cliente: 127.0.0.1 (localhost)
  • dhclient eth0 para configurar o ip na interface de rede eth0 automaticamente

Para facilitar, depois de conseguir o IP do CLiente, vá no arquivo LDR e coloque o IP na variável cliente_ip. Com essa mudança, todas as vezes que esse IP aparecer nessa página, ele será o IP do seu Cliente.

Obtendo IP do SERVIDOR

Precisamos primeiramente estar como Super Usuário.

No terminal, digite:

  1. sudo su para mudar o terminal para o super usuário
  2. senha: 654123
  3. ifconfig para listar os ips da maquina
  4. Pegue o IP que vem após o eth0 no resultado do ifconfig
  • Exemplo: 192.168.0.112

Para facilitar, depois de conseguir o IP do CLiente, vá no arquivo LDR e coloque o IP na variável server_ip. Com essa mudança, todas as vezes que esse IP aparecer nessa página, ele será o IP do seu Servidor.
